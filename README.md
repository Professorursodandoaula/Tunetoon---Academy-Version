# Tunetoon - Academy Version

Multi-toon launcher for Toontown Rewritten with Multicontroller integration.

## Features
- Login and management of multiple accounts
- Integrated Multicontroller with automatic Mirror Mode and Mouse Replication
- Automatic toon distribution by groups based on window layout
- SOS Card Detector
- Boss Calculator (CJ & CEO)
- Ctrl+M shortcut to open/close the Multicontroller
- Multi-monitor support

## How to use
1. Download the latest release
2. Extract the "Tunetoon - Academy Version" folder
3. Run Tunetoon.exe
